## What is Venti?

벤티는 이벤트 및 할인 정보 모음 플랫폼으로, 유저에게 맞춤형 이벤트 추천 기능과 검색 기능, 알림 기능 등을 제공합니다.  
[VENTI 홈페이지 바로가기](https://m.venti.life/)
